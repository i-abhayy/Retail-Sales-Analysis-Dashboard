# Retail Sales Analysis Dashboard

A complete Retail Sales Analysis project built using SQL, Excel, and Power BI to analyze sales performance, customer behavior, product trends, and business insights.

## Tools & Technologies

* SQL (Data Cleaning, Querying & Analysis)
* Microsoft Excel (Data Preparation & Validation)
* Power BI (Interactive Dashboards & Data Visualization)

## Key Features

* Sales Performance Analysis
* Customer Analysis Dashboard
* Product Analysis Dashboard
* Revenue and Profit Insights
* Regional Sales Trends
* Discount Impact Analysis
* Interactive KPIs and Visualizations

## Project Highlights

* Cleaned and transformed raw retail sales data using SQL and Excel.
* Created analytical queries to extract business insights.
* Designed interactive Power BI dashboards for decision-making.
* Identified top-performing products, customers, and sales trends.

This project demonstrates end-to-end data analytics skills, from data extraction and transformation to visualization and business intelligence reporting.
